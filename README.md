# ClearFPS
ClearFps: Otimização de performance de cliente para servidores Minecraft de alta escala. Um plugin de gestão de renderização que permite aos jogadores isolar, ocultar entidades e gerir condições ambientais para maximizar quadros por segundo (FPS) em cenários de alta carga de entidades.
